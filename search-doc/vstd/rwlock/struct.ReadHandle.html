Struct ReadHandle

pub struct ReadHandle<'a, V, Pred: RwLockPredicate<V>> { /* private fields */ }
Expand description

Handle obtained for a shared read-lock from an RwLock.

Warning: The lock is NOT released automatically when the handle is dropped. You must call release_read. Verus does not check that lock is released.
